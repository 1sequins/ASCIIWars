Подробное описание

Представляет игрока.

У игрока есть такие свойства:

  • здоровье (health)
    • Ограничено максимальным значением (maxHealth), которое может увеличиваться (с помощью ASCIIWars.Game.HealthBooster)
  • атака (attack)
  • защита (defense)
  • инвентарь (inventory)
    • Размер ограничнен максимальным значением (MAX_INVENTORY_SIZE), которое не может увеличиваться
  • монеты (coins)
    • В начале игры даётся определённое количиство монет (STARTER_COINS)
