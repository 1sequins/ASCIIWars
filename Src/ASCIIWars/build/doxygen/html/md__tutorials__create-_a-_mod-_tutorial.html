В игре есть уже встроенная поддержка моддинга. Принцип создания мода довольно простой, о этом и пойдёт речь в этой главе.

  2. Добавляем в зависимости решение ASCIIWars. Не знаю как, но это обязательно надо сделать. Хотя, есть один способ:

Отлично! теперб можно использовать классы из namespace'а ASCIIWars!

  1. Теперь нужно скопировать и вставить вот этот код в главный файл:

    using ASCIIWars.Modding;
    public class MyMod : ModDescriptor {
    public MyMod(ModInfo modInfo) : base(modInfo) { }
    public override void OnLoadBy(ModLoader modLoader) {
    }
    public override void OnDestroyBy(ModLoader modLoader) {
    }
    }

    Метод OnLoadBy вызывается после того, как мод был загружен. Метод OnDestroyBy вызывается после того, как приложение завершило работу.
