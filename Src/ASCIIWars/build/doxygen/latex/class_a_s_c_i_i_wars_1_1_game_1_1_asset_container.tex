\subsection{Подробное описание}
Контейнер ассетов. Хранит их по групам. В каждой группе находятся файлы ассетов. 

В представлении файловой системы выглядит так\+: 
\begin{DoxyCode}
> assetsDirectory/
>     assetGroup1/
>         asset1.txt
>         asset2.json
>     assetGroup2/
>         asset1.png
>         asset2.mp3
\end{DoxyCode}
